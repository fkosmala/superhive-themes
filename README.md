# superhive-themes

This repository host few themes for [SuperHive](https://github.com/fkosmala/superhive) Blockchained Blog Engine.

## Install
Just go into the `/public/` folder on your SuperHive instance and clone this repo :

```sh
git clone https://github.com/fkosmala/superhive-themes.git themes
```

And that's all !
